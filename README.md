# Data Science Internship: Task-01
## Population Distribution Analysis

### Project Overview
This project involves visualizing the distribution of categorical and continuous variables within a population dataset. Using the **Titanic Dataset**, I have created visualizations to understand the demographics of the passengers, specifically focusing on **Gender** (Categorical) and **Age** (Continuous).

### Key Objectives
* Perform data cleaning and handle missing values (specifically in the Age column).
* Visualize the distribution of categorical variables using **Bar Charts**.
* Visualize the distribution of continuous variables using **Histograms** with Kernel Density Estimates (KDE).

### Dataset
The dataset used is `Data.csv`, which contains information about passengers, including:
* **Sex**: Categorical variable (Male/Female).
* **Age**: Continuous variable (Numerical).
* **Survived**: Binary classification.
* **Pclass**: Socio-economic status.

### Technologies Used
* **Python 3.13**
* **Pandas**: Data manipulation and analysis.
* **Matplotlib**: Core plotting library.
* **Seaborn**: Statistical data visualization.

### Visualizations

#### 1. Gender Distribution (Bar Chart)
The bar chart illustrates the count of passengers categorized by gender. This helps in understanding the gender ratio within the population sample.

#### 2. Age Distribution (Histogram)
The histogram displays the frequency of various age groups. The inclusion of a KDE curve helps identify the density and peaks within the age groups (e.g., a high concentration of passengers aged 20-40).
